    如何对linux程序进行性能分析

    Samson Wang
    2018-06-13 17:46

    Source

      Tags :
    • linux
    • profiling

    对既有程序进行性能分析能够找出程序中性能瓶颈的代码，为后续的优化工作起到指导作用。

    使用基本的命令行工具抓取程序的运行状态

    查看程序的调用堆栈
    pstack 命令

    pstack <pid>

    查看程序的运行状态
    top 命令
    使用 -b 命令能够让 top 命令批量执行

    top -b -p <pid>

    ps 命令
    使用 format 参数能够设置输出的内容，这样就能够查看进程的CPU和内存的使用情况。

    ps --format pid,pcpu,cputime,etime,size,vsz,cmd -p <pid>

    使用perf对程序进行性能分析

    专业的性能评估工具（linux）

    gprof
    oprof
