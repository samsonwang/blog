#+TITLE: Linux信号概览

在linux系统的信号是很重要的

可以使用 =man 7 signal= 查看操作系统中所有的信号


| signal | value | action | comment         |
|--------+-------+--------+-----------------|
| SIGHUP |     1 | term   | hangup detected |
|        |       |        |                 |
